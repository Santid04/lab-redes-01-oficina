# Lab 1 – Red básica de oficina

## Descripción
Este laboratorio consiste en la creación de una red local simple simulando una oficina pequeña, utilizando Cisco Packet Tracer.

La red permite la comunicación entre múltiples equipos dentro de la misma red utilizando direccionamiento IPv4 estático.

## Topología
- 1 Switch Cisco 2960
- 4 PCs conectadas al switch

## Direccionamiento IP
- PC0: 192.168.1.10 / 255.255.255.0
- PC1: 192.168.1.11 / 255.255.255.0
- PC2: 192.168.1.12 / 255.255.255.0
- PC3: 192.168.1.13 / 255.255.255.0

No se configura gateway ya que no se utiliza router en este laboratorio.

## Pruebas de conectividad
Se realizaron pruebas de conectividad mediante el comando `ping` entre los distintos equipos, verificando comunicación exitosa dentro de la red local.

## Archivos incluidos
